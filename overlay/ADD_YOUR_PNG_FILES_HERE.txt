NAMING TIPS:
------------

SPECIFICATIONS:
---------------
- Recommended size: 64x64 or 96x96 pixels
- Format: PNG with transparent background
- No limit on number of items!

CUSTOM APP ICON:
----------------
To give the ChatCollect.exe a custom icon:
1. Create an .ico file (you can convert a PNG to ICO online).
2. Name it "exe_icon.ico".
3. Place it in the main folder (where the .exe is).
4. Run "build_exe.bat" again.
